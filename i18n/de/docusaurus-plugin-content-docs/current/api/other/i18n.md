# i18n

### Description

@short: Eine Sammlung von Methoden zur Lokalisierung des Schedulers

@signature: i18n: any


### Details

Das **i18n**-Objekt enthält folgende Methoden:

- **addLocale(languageCode, localeObject)** - ermöglicht das Hinzufügen eines benutzerdefinierten Locale zum Scheduler
    - **languageCode** - (*string*) ein zweibuchstabiger Sprachcode
    - **localeObject** - (*object*) das Locale-Datenobjekt
- **setLocale(localeObject: string|object)** - setzt das aktive Locale

Sie können entweder einen Sprachcode übergeben:

~~~js
scheduler.i18n.setLocale("fr");
~~~

oder ein Locale-Objekt:

~~~js
scheduler.i18n.setLocale({
       labels: {
        day_tab: "Day",
   }
});
~~~

- **getLocale(languageCode : string)** - ruft ein Locale-Objekt anhand seines Sprachcodes ab
