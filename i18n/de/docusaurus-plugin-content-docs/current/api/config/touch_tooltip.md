# touch_tooltip

### Description

@short: Ermöglicht das Anzeigen von dhtmlxScheduler-Tooltips auf Touch-Geräten

@signature: touch_tooltip: boolean

### Example

~~~jsx
// aktiviert Tooltips für Touch-Geräte
scheduler.config.touch_tooltip = true;
~~~

**Default value:** false
