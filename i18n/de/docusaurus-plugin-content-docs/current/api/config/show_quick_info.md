# show_quick_info

### Description

@short: Aktiviert oder deaktiviert die 'quick_info' Erweiterung (das Pop-up, das die Aufgabendetails anzeigt)

@signature: show_quick_info: boolean

### Example

~~~jsx
scheduler.config.show_quick_info = false;
scheduler.init("scheduler_here");
~~~

**Default value:** true
