# collapse

### Description

@short: Gibt den Scheduler von seinem erweiterten Zustand zurück zur regulären Größe

@signature: collapse: () =\> void

### Example

~~~jsx
scheduler.collapse();
~~~

### Details

:::note
 Die Methode benötigt die [expand](guides/extensions-list.md#expand) Erweiterung, um aktiv zu sein. 
:::
