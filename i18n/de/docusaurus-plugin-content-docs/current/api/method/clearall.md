# clearAll

### Description

@short: Entfernt alle Events aus dem Scheduler

@signature: clearAll: () =\> void

### Example

~~~jsx
scheduler.clearAll();
~~~

### Details

Beachten Sie, dass diese Methode das Event [onClearAll](api/event/onclearall.md) auslöst.
