# setLightboxSize

### Description

@short: Aktualisiert die Größe der Lightbox

@signature: setLightboxSize: () =\> void

### Example

~~~jsx
var control = scheduler.formSection("description");
control.header.style.display = "none";

scheduler.setLightboxSize();
~~~

### Details

Diese Methode ist nützlich, um die Größe der Lightbox anzupassen, nachdem bestimmte Abschnitte ausgeblendet oder angezeigt wurden.
