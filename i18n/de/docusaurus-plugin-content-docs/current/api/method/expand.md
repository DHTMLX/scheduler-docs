# expand

### Description

@short: Öffnet den Scheduler, sodass er den gesamten Bildschirm für bessere Sichtbarkeit einnimmt

@signature: expand: () =\> void

### Example

~~~jsx
scheduler.expand();
~~~

### Details

:::note
 Die Methode erfordert das aktivierte [expand](guides/extensions-list.md#expand) Plugin. 
:::
