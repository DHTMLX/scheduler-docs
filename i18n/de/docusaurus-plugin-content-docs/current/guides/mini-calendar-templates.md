# Mini-Kalender-Vorlagen

![mini_calendar_templates](/img/mini_calendar_templates.png)

- [calendar_date](api/template/calendar_date.md) - definiert, wie der Tag in jeder Zelle dargestellt wird
- [calendar_month](api/template/calendar_month.md) - legt das Datumsformat fest, das im Kalender-Header angezeigt wird
- [calendar_scale_date](api/template/calendar_scale_date.md) - steuert die angezeigten Tagesnamen in der Wochenunterüberschrift
- [calendar_time](api/template/calendar_time.md) - gibt das Datumsformat für die Start- und Enddatumsfelder im Lightbox-Dialog an
