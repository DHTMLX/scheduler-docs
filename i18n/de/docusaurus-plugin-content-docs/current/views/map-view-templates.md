# Map-Ansichtsvorlagen

![map_view_templates](/img/map_view_templates.png)

- [map_date](api/template/map_date.md) - zeigt das Datum in der Kopfzeile der Ansicht an
- [map_text](api/template/map_text.md) - zeigt den Text in der zweiten Spalte der Ansicht an
- [map_time](api/template/map_time.md) - präsentiert das Datum in der ersten Spalte der Ansicht
- [map_info_content](api/template/map_info_content.md) - definiert den Inhalt des Info-Fensters in der Kartenansicht
