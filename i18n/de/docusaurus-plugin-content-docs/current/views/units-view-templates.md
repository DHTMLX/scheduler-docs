# Units View Templates

![units_view_templates](/img/units_view_templates.png)

- [event_header](api/template/event_header.md) - definiert den Ereignis-Header
- [event_text](api/template/event_text.md) - steuert den Inhalt des Ereignistexts
- [event_class](api/template/event_class.md) - legt den CSS-Stil für den Ereigniscontainer fest
- [hour_scale](api/template/hour_scale.md) - stellt die Elemente auf der Y-Achse dar
- [week_date_class](api/template/week_date_class.md) - bestimmt die CSS-Klasse, die auf eine Tageszelle angewendet wird
- [](api/template/unitsname_date.md) - zeigt das Datum in der Kopfzeile der Ansicht an
- [](api/template/unitsname_scale_text.md) - zeigt die Elemente auf der X-Achse an
