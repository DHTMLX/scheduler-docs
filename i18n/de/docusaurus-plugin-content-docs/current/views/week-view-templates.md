# Week-View-Vorlagen

![week_view_templates](/img/week_view_templates.png)

- [event_class](api/template/event_class.md) - definiert den CSS-Stil für den Ereignis-Container
- [event_header](api/template/event_header.md) - stellt den Kopfbereich eines Ereignisses dar
- [event_text](api/template/event_text.md) - zeigt den Haupttextinhalt eines Ereignisses an
- [hour_scale](api/template/hour_scale.md) - zeigt die Elemente entlang der Y-Achse an
- [week_date](api/template/week_date.md) - präsentiert das Datum in der Kopfzeile der Ansicht
- [week_date_class](api/template/week_date_class.md) - legt den CSS-Stil für einzelne Tageszellen fest
- [week_scale_date](api/template/week_scale_date.md) - zeigt das Datum in der Unterkopfzeile der Ansicht an
