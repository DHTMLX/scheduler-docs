# Day-Ansicht Vorlagen

![day_view_templates](/img/day_view_templates.png)

- [day_date](api/template/day_date.md) - zeigt das Datum in der Kopfzeile an  
- [day_scale_date](api/template/day_scale_date.md) - zeigt das Datum in der Unterkopfzeile an  
- [event_class](api/template/event_class.md) - definiert den CSS-Stil, der auf den Ereignis-Container angewendet wird  
- [event_header](api/template/event_header.md) - repräsentiert den Kopfbereich eines Ereignisses  
- [event_text](api/template/event_text.md) - enthält den Textinhalt eines Ereignisses  
- [hour_scale](api/template/hour_scale.md) - rendert die Elemente entlang der Y-Achse
