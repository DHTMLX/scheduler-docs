# Month View Templates

![month_view_templates](/img/month_view_templates.png)

- [event_bar_date](api/template/event_bar_date.md) - zeigt das Datum eines Ereignisses an, speziell für eintägige Ereignisse verwendet
- [event_bar_text](api/template/event_bar_text.md) - zeigt den Text des Ereignisses an, gilt für alle Ereignisse
- [event_class](api/template/event_class.md) - definiert den CSS-Stil für den Ereigniscontainer, wird sowohl für eintägige als auch mehrtägige Ereignisse verwendet
- [month_date](api/template/month_date.md) - stellt das Datum dar, das in der Kopfzeile der Ansicht angezeigt wird
- [month_date_class](api/template/month_date_class.md) - wendet CSS-Stile auf einzelne Tageszellen an
- [month_day](api/template/month_day.md) - steuert das Format des innerhalb einer Zelle angezeigten Tages
- [month_events_link](api/template/month_events_link.md) - verwaltet den „Mehr anzeigen"-Link innerhalb einer Zelle, verwendet für eintägige und mehrtägige Ereignisse
- [month_scale_date](api/template/month_scale_date.md) - legt die Tagesnamen in der Wochenkopfzeile der Ansicht fest
