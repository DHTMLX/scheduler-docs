# Grid-View-Vorlagen

![grid_view_templates](/img/grid_view_templates.png)

- [event_class](api/template/event_class.md) - definiert den CSS-Stil, der auf die Grid-Zeile angewendet wird
- [`{gridname}_date_template`](api/template/gridname_date.md) - steuert die Datumsanzeige in der Kopfzeile der Ansicht
- [`{gridname}_full_date_template`](api/template/gridname_full_date.md) - legt das Datumsformat für Spalten mit id='date' fest
- [`{gridname}_single_date_template`](api/template/gridname_single_date.md) - bestimmt das Datumsformat für Spalten mit id='start_date' oder id='end_date'
- [`{gridname}_field_template`](api/template/gridname_field.md) - bestimmt den Text, der in den Spalten des Grids angezeigt wird
