# Agenda-Ansicht-Vorlagen

*Für Nutzer von dhtmlxScheduler 6.0 oder älter sind zusätzliche Informationen [hier](views/agenda-view-templates-legacy.md) verfügbar.

- [agenda_date](api/template/agenda_date.md) - definiert das Datum, das in der Kopfzeile der Ansicht angezeigt wird
- [agenda_text](api/template/agenda_text.md) - steuert, wie der Ereignistitel dargestellt wird
- [agenda_time](api/template/agenda_time.md) - legt das Format für die Ereigniszeit fest
- [agenda_day](api/template/agenda_day.md) - passt das Aussehen des Datumslabels an
