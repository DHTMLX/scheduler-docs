# WeekAgenda-Ansichtsvorlagen

![view_templates_week_agenda](/img/view_templates_week_agenda.png)

- [week_agenda_event_text](api/template/week_agenda_event_text.md) - definiert den Text, der für jedes Ereignis angezeigt wird
- [week_agenda_scale_date](api/template/week_agenda_scale_date.md) - legt das Datum fest, das in jeder Tageszelle der Ansicht angezeigt wird
- [week_date](api/template/week_date.md) - steuert das Datum, das in der Kopfzeile der Ansicht angezeigt wird
- [week_agenda_date](api/template/week_agenda_date.md) - bestimmt das Datum, das in der Kopfzeile der Week Agenda-Ansicht angezeigt wird
