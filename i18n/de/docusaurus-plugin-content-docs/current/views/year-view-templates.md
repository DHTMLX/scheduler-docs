# Jahresansicht-Vorlagen

![year_view_templates](/img/year_view_templates.png)

- [event_class](api/template/event_class.md) - definiert das CSS-Styling für den Ereignis-Container
- [month_day](api/template/month_day.md) - legt fest, wie der Tag innerhalb einer Zelle dargestellt wird
- [year_date](api/template/year_date.md) - zeigt das Datum in der Kopfzeile der Ansicht an
- [year_month](api/template/year_month.md) - zeigt den Monatsnamen in der Kopfzeile jedes Monatsblocks an
- [year_scale_date](api/template/year_scale_date.md) - präsentiert den Tagesnamen in der Unterkopfzeile innerhalb eines Monatsblocks
- [year_tooltip](api/template/year_tooltip.md) - stellt das Tooltip für eine Tageszelle bereit, die geplante Ereignisse enthält
