# Timeline-Ansichtsvorlagen

![timeline_view_templates_01](/img/timeline_view_templates_01.png)


![timeline_view_templates_02](/img/timeline_view_templates_02.png)

- [event_bar_text](api/template/event_bar_text.md) - zeigt den Text innerhalb eines Ereignisses an
- [event_class](api/template/event_class.md) - definiert den CSS-Stil für den Ereignis-Container
- [`TIMELINE_cell_value`](api/template/timelinename_cell_value.md) - zeigt die Anzahl der geplanten Ereignisse innerhalb einer Zelle der Ansicht an
- [`TIMELINE_cell_class`](api/template/timelinename_cell_class.md) - wendet den CSS-Stil auf eine Zelle an
- [`TIMELINE_scalex_class`](api/template/timelinename_scalex_class.md) - legt den CSS-Stil für Elemente auf der X-Achse fest
- [`TIMELINE_second_scalex_class`](api/template/timelinename_second_scalex_class.md) - gestaltet die Elemente auf der zweiten X-Achse mit CSS
- [`TIMELINE_scaley_class`](api/template/timelinename_scaley_class.md) - steuert den CSS-Stil für Elemente auf der Y-Achse
- [`TIMELINE_scale_label`](api/template/timelinename_scale_label.md) - stellt die Beschriftungen auf der Y-Achse dar
- [`TIMELINE_tooltip`](api/template/timelinename_tooltip.md) - zeigt das Tooltip für eine Zelle mit einem oder mehreren geplanten Ereignissen an
- [`TIMELINE_date`](api/template/timelinename_date.md) - zeigt das Datum in der Kopfzeile der Ansicht an
- [`TIMELINE_row_class`](api/template/timelinename_row_class.md) - gibt die CSS-Klasse an, die einer Zeile in der Ansicht zugewiesen wird
- [`TIMELINE_scale_date`](api/template/timelinename_scale_date.md) - stellt die Elemente auf der X-Achse dar
- [`TIMELINE_second_scale_date`](api/template/timelinename_second_scale_date.md) - stellt die Elemente auf der zweiten X-Achse dar
