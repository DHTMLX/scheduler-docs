# Agenda-Ansichtsvorlagen

*Dieser Artikel bezieht sich auf dhtmlxScheduler 6.0 und frühere Versionen. Informationen zu dhtmlxScheduler 7.0 und höher finden Sie [hier](views/agenda-view-templates.md).*

![agenda_view_templates](/img/agenda_view_templates.png)

- [agenda_date](api/template/agenda_date.md) - steuert das im Kopfbereich der Ansicht angezeigte Datum  
- [agenda_text](api/template/agenda_text.md) - definiert den Text, der in der zweiten Spalte angezeigt wird  
- [agenda_time](api/template/agenda_time.md) - legt das Datumsformat für die erste Spalte fest
