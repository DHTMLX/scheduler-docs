# Спецификация формата даты

При настройке форматов даты вы можете использовать любые из следующих символов:


- **%y** - обозначает год в виде двухзначного числа ( _00 до 99_ );
- **%Y** - обозначает год в виде четырехзначного числа ( _1900-9999_ );


- **%m** - месяц в виде двухзначного числа с ведущим нулём ( _01 до 12_ );
- **%n** - месяц в виде числа без ведущего нуля ( _1 до 12_ );
- **%M** - сокращённое название месяца ( _Jan до Dec_ );
- **%F** - полное название месяца ( _January до December_ );


- **%W** - номер недели года по ISO-8601, недели начинаются с понедельника; 


- **%d** - день месяца в виде двухзначного числа с ведущим нулём ( _01 до 31_ );
- **%j** - день месяца в виде числа без ведущего нуля ( _1 до 31_ );
- **%D** - сокращённое название дня недели ( _Sun до Sat_ );
- **%l** - полное название дня недели ( _Sunday до Saturday_ );


- **%h** - часы по 12-часовому формату ( _00 до 11_ );
- **%H** - часы по 24-часовому формату ( _00 до 23_ );
- **%g** - часы по 12-часовому формату без ведущего нуля ( _1 до 12_ );
- **%G** - часы по 24-часовому формату без ведущего нуля ( _0 до 23_ );


- **%i** - минуты в виде двухзначного числа с ведущим нулём ( _00 до 59_ );
- **%s** - секунды в виде двухзначного числа с ведущим нулём ( _00 до 59_ );
- **%a** - отображает **am** (для времени с полуночи до полудня) и **pm** (для времени с полудня до полуночи);
- **%A** - отображает **AM** (для времени с полуночи до полудня) и **PM** (для времени с полудня до полуночи).


Например, чтобы отобразить 1 июня 2013 года как 01/06/2013, используйте формат "%d/%m/%Y".
