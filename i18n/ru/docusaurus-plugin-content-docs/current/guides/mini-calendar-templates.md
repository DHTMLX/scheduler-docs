# Шаблоны мини-календаря

![mini_calendar_templates](/img/mini_calendar_templates.png)

- [calendar_date](api/template/calendar_date.md) - определяет, как отображается день внутри каждой ячейки
- [calendar_month](api/template/calendar_month.md) - задает формат даты, отображаемый в заголовке календаря
- [calendar_scale_date](api/template/calendar_scale_date.md) - управляет названиями дней, отображаемыми в подзаголовке недели
- [calendar_time](api/template/calendar_time.md) - указывает формат даты, используемый для полей ввода даты начала и окончания в lightbox
