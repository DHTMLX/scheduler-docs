# section_delimiter

### Description

@short: Определяет разделитель, используемый для разделения нескольких секций или единиц внутри связанного свойства данных события

@signature: section_delimiter: string

### Example

~~~jsx
scheduler.config.section_delimiter = ";";
~~~

**Default value:** ","
