# touch_swipe_dates

### Description

@short: Позволяет включить или отключить возможность переключения активных дат с помощью горизонтального жеста swipe на мобильных устройствах

@signature: touch_swipe_dates: boolean

### Example

~~~jsx
scheduler.config.touch_swipe_dates = true;
~~~

**Default value:** false
