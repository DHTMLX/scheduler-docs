# touch_swipe_dates

### Description

@short: Включает/выключает переключение активных дат с помощью горизонтального жеста свайпа на мобильных устройствах

@signature: touch_swipe_dates: boolean

### Example

~~~jsx
scheduler.config.touch_swipe_dates = true;
~~~

**Значение по умолчанию:** false
