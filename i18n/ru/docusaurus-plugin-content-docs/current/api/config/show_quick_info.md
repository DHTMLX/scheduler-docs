# show_quick_info

### Description

@short: Включает или отключает расширение 'quick_info' (всплывающее окно, показывающее детали задачи)

@signature: show_quick_info: boolean

### Example

~~~jsx
scheduler.config.show_quick_info = false;
scheduler.init("scheduler_here");
~~~

**Default value:** true
