# show_errors

### Description

@short: Включает всплывающие окна с предупреждениями об ошибках при возникновении проблем

@signature: show_errors: boolean

### Example

~~~jsx
scheduler.config.show_errors = false;
~~~

**Значение по умолчанию:** true
