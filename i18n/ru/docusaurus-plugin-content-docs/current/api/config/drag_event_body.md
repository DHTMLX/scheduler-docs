# drag_event_body

### Description

@short: Позволяет перетаскивать события расписания по любой части тела события

@signature: drag_event_body: boolean

### Example

~~~jsx
scheduler.config.drag_event_body = false;
~~~

**Значение по умолчанию:** true

### Related samples
- [Основная инициализация](https://docs.dhtmlx.com/scheduler/samples/01_initialization_loading/01_basic_init.html)

### Details

Если конфигурация отключена, перетаскивание возможно только по заголовку события.
