# touch_tooltip

### Description

@short: Включает отображение тултипов dhtmlxScheduler на устройствах с сенсорным экраном

@signature: touch_tooltip: boolean

### Example

~~~jsx
// включает тултипы для сенсорных устройств
scheduler.config.touch_tooltip = true;
~~~

**Default value:** false
