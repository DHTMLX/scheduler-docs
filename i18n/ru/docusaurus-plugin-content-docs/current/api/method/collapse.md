# collapse

### Description

@short: Сворачивает развернутый планировщик обратно до обычного размера

@signature: collapse: () =\> void

### Example

~~~jsx
scheduler.collapse();
~~~

### Details

:::note
 Метод требует, чтобы расширение [expand](guides/extensions-list.md#expand) было включено. 
:::
