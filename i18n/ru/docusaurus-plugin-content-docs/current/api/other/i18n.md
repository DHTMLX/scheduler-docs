# i18n

### Description

@short: Набор методов для локализации Scheduler

@signature: i18n: any


### Details

Объект **i18n** включает следующие методы:

- **addLocale(languageCode, localeObject)** - позволяет добавить пользовательскую локаль в Scheduler
    - **languageCode** - (*string*) двухбуквенный код языка
    - **localeObject** - (*object*) объект с данными локали
- **setLocale(localeObject: string|object)** - устанавливает активную локаль

Вы можете передать либо код языка:

~~~js
scheduler.i18n.setLocale("fr");
~~~

либо объект локали:

~~~js
scheduler.i18n.setLocale({
       labels: {
        day_tab: "Day",
   }
});
~~~

- **getLocale(languageCode : string)** - возвращает объект локали по коду языка
