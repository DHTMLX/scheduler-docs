# Сторонние лицензии

В этой статье перечислены уведомления и дополнительные условия для сторонних программных компонентов, включённых в библиотеку DHTMLX Scheduler.

**Шрифт Roboto от Google** по умолчанию не входит в состав библиотеки. Начиная с версии Scheduler v5.0, тема Material динамически загружает шрифт Roboto с `https://fonts.googleapis.com`. Этот шрифт лицензирован по лицензии Apache Software License.

Библиотека DHTMLX Scheduler предоставляет исходные LESS-файлы для оформления Scheduler. Вы можете установить NPM-пакет и использовать скрипт npm-watch из файла *codebase/sources/less/package.json* для настройки и пересборки скинов. Библиотеки **npm-watch** и **less**, используемые для этих целей, лицензированы соответственно по MIT и Apache лицензиям и по умолчанию не включены в пакет Scheduler.

Кроме того, в демонстрационных приложениях используются некоторые open source библиотеки, такие как **DHTMLX Suite 5.x** и **DHTMLX Suite 6.x**. В примерах также используются модули Node.js **Underscore** и **Backbone**. Библиотека **jQuery** загружается с CDN. В серверных примерах используются модули Node.js, такие как **Express, body-parser, helmet** и другие.
