# Шаблоны годового вида

![year_view_templates](/img/year_view_templates.png)

- [event_class](api/template/event_class.md) - определяет CSS-стили для контейнера события
- [month_day](api/template/month_day.md) - задаёт отображение дня внутри ячейки
- [year_date](api/template/year_date.md) - отображает дату в заголовке вида
- [year_month](api/template/year_month.md) - показывает название месяца в заголовке каждого блока месяца
- [year_scale_date](api/template/year_scale_date.md) - выводит название дня недели в подзаголовке внутри блока месяца
- [year_tooltip](api/template/year_tooltip.md) - предоставляет тултип для ячейки дня, содержащей запланированные события
