# Шаблоны представления грида

![grid_view_templates](/img/grid_view_templates.png)

- [event_class](api/template/event_class.md) - определяет CSS-стиль, применяемый к строке грида
- [`{gridname}_date_template`](api/template/gridname_date.md) - управляет отображением даты в заголовке представления
- [`{gridname}_full_date_template`](api/template/gridname_full_date.md) - задаёт формат даты, используемый в колонках с id='date'
- [`{gridname}_single_date_template`](api/template/gridname_single_date.md) - устанавливает формат даты для колонок с id='start_date' или id='end_date'
- [`{gridname}_field_template`](api/template/gridname_field.md) - определяет текст, отображаемый в колонках грида
