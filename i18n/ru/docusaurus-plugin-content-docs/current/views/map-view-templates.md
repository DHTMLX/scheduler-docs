# Шаблоны Map View

![map_view_templates](/img/map_view_templates.png)

- [map_date](api/template/map_date.md) - отображает дату в заголовке представления
- [map_text](api/template/map_text.md) - показывает текст во второй колонке представления
- [map_time](api/template/map_time.md) - выводит дату в первой колонке представления
- [map_info_content](api/template/map_info_content.md) - определяет содержимое информационного окна в Map View
