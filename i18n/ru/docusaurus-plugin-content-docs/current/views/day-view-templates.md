# Шаблоны для дневного вида

![day_view_templates](/img/day_view_templates.png)

- [day_date](api/template/day_date.md) - отображает дату в заголовке  
- [day_scale_date](api/template/day_scale_date.md) - показывает дату в подзаголовке  
- [event_class](api/template/event_class.md) - определяет CSS-стиль, применяемый к контейнеру события  
- [event_header](api/template/event_header.md) - представляет секцию заголовка события  
- [event_text](api/template/event_text.md) - содержит текстовое содержимое события  
- [hour_scale](api/template/hour_scale.md) - отображает элементы вдоль оси Y
