# Шаблоны представления Timeline

![timeline_view_templates_01](/img/timeline_view_templates_01.png)

![timeline_view_templates_02](/img/timeline_view_templates_02.png)

- [event_bar_text](api/template/event_bar_text.md) - отображает текст внутри события
- [event_class](api/template/event_class.md) - определяет CSS-стиль для контейнера события
- [`TIMELINE_cell_value`](api/template/timelinename_cell_value.md) - показывает количество запланированных событий внутри ячейки представления
- [`TIMELINE_cell_class`](api/template/timelinename_cell_class.md)  - применяет CSS-стиль к ячейке
- [`TIMELINE_scalex_class`](api/template/timelinename_scalex_class.md) - задаёт CSS-стиль для элементов по оси X
- [`TIMELINE_second_scalex_class`](api/template/timelinename_second_scalex_class.md) - стилизует элементы по второй оси X с помощью CSS
- [`TIMELINE_scaley_class`](api/template/timelinename_scaley_class.md) - управляет CSS-стилем для элементов по оси Y
- [`TIMELINE_scale_label`](api/template/timelinename_scale_label.md) - отображает подписи по оси Y
- [`TIMELINE_tooltip`](api/template/timelinename_tooltip.md)  - показывает тултип для ячейки, содержащей одно или несколько запланированных событий
- [`TIMELINE_date`](api/template/timelinename_date.md) - отображает дату в заголовке представления
- [`TIMELINE_row_class`](api/template/timelinename_row_class.md) - указывает CSS-класс, назначаемый строке в представлении
- [`TIMELINE_scale_date`](api/template/timelinename_scale_date.md) - отображает элементы по оси X
- [`TIMELINE_second_scale_date`](api/template/timelinename_second_scale_date.md) - отображает элементы по второй оси X
