# Шаблоны Units View

![units_view_templates](/img/units_view_templates.png)

- [event_header](api/template/event_header.md) - определяет заголовок события
- [event_text](api/template/event_text.md) - отвечает за содержимое текста события
- [event_class](api/template/event_class.md) - задаёт CSS-стиль для контейнера события
- [hour_scale](api/template/hour_scale.md) - отображает элементы на оси Y
- [week_date_class](api/template/week_date_class.md) - определяет CSS-класс, применяемый к ячейке дня
- [](api/template/unitsname_date.md) - отображает дату в заголовке представления
- [](api/template/unitsname_scale_text.md) - показывает элементы на оси X
