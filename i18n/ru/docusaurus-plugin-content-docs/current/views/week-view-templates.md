# Шаблоны недельного вида

![week_view_templates](/img/week_view_templates.png)

- [event_class](api/template/event_class.md) - определяет CSS-стиль для контейнера события
- [event_header](api/template/event_header.md) - отображает заголовок события
- [event_text](api/template/event_text.md) - показывает основной текст события
- [hour_scale](api/template/hour_scale.md) - отображает элементы вдоль оси Y
- [week_date](api/template/week_date.md) - отображает дату в заголовке вида
- [week_date_class](api/template/week_date_class.md) - задаёт CSS-стиль для отдельных ячеек дней
- [week_scale_date](api/template/week_scale_date.md) - показывает дату в подзаголовке вида
