# Шаблоны представления Agenda

*Эта статья относится к dhtmlxScheduler версии 6.0 и ниже. Для информации о dhtmlxScheduler 7.0 и выше, пожалуйста, смотрите [здесь](views/agenda-view-templates.md).*

![agenda_view_templates](/img/agenda_view_templates.png)

- [agenda_date](api/template/agenda_date.md) - управляет датой, отображаемой в заголовке представления  
- [agenda_text](api/template/agenda_text.md) - определяет текст, отображаемый во втором столбце  
- [agenda_time](api/template/agenda_time.md) - задает формат даты для первого столбца
