# Шаблоны для Месячного Вида

![month_view_templates](/img/month_view_templates.png)

- [event_bar_date](api/template/event_bar_date.md) - отображает дату события, используется специально для однодневных событий
- [event_bar_text](api/template/event_bar_text.md) - отображает текст события, применяется ко всем событиям
- [event_class](api/template/event_class.md) - определяет CSS-стиль для контейнера события, используется как для однодневных, так и для многодневных событий
- [month_date](api/template/month_date.md) - представляет дату, отображаемую в заголовке вида
- [month_date_class](api/template/month_date_class.md) - применяет CSS-стилизацию к отдельным ячейкам дней
- [month_day](api/template/month_day.md) - управляет форматом дня, отображаемого внутри ячейки
- [month_events_link](api/template/month_events_link.md) - управляет ссылкой 'View more' внутри ячейки, используется как для однодневных, так и для многодневных событий
- [month_scale_date](api/template/month_scale_date.md) - задаёт названия дней недели в заголовке вида
