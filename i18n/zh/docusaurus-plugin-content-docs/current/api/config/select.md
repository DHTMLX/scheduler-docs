# select

### Description

@short: 切换事件框内select栏的可见性

@signature: select: boolean

### Example

~~~jsx
scheduler.config.select = false;
...
scheduler.init('scheduler_here',new Date(2013,0,10),"week");
~~~

**Default value:** true
