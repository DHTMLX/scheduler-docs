# show_quick_info

### Description

@short: 开启或关闭 'quick_info' 扩展（显示任务详情的弹出框）

@signature: show_quick_info: boolean

### Example

~~~jsx
scheduler.config.show_quick_info = false;
scheduler.init("scheduler_here");
~~~

**Default value:** true
