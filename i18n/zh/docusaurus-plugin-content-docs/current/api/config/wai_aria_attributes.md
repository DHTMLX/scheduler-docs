# wai_aria_attributes

### Description

@short: 启用 WAI-ARIA 支持，使组件能够被屏幕阅读器识别

@signature: wai_aria_attributes: boolean

### Example

~~~jsx
scheduler.config.wai_aria_attributes = true;
...
scheduler.init("scheduler_here");
~~~

**Default value:** true

### Details

在版本 4.4 中引入
