# touch_swipe_dates

### Description

@short: 允许您开启或关闭在移动设备上通过水平滑动手势切换活动日期的功能

@signature: touch_swipe_dates: boolean

### Example

~~~jsx
scheduler.config.touch_swipe_dates = true;
~~~

**Default value:** false
