# touch_tooltip

### Description

@short: 使 dhtmlxScheduler 在触摸设备上显示 tooltips

@signature: touch_tooltip: boolean

### Example

~~~jsx
// 开启触摸设备上的 tooltips
scheduler.config.touch_tooltip = true;
~~~

**Default value:** false
