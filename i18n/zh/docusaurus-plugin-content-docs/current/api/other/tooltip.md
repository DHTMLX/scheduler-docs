# tooltip

### Description

@short: 显示事件的tooltip

@signature: tooltip: any

### Example

~~~jsx
var tooltip = scheduler.tooltip;
tooltip.hide();
tooltip.show(event, text);
~~~

### Details

:::note
 该选项仅在激活了[tooltip](guides/extensions-list.md#tooltip) 插件时有效。 
:::
