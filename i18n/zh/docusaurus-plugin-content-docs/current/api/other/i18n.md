# i18n

### Description

@short: Scheduler 本地化方法集合

@signature: i18n: any


### Details

**i18n** 对象包含以下方法:

- **addLocale(languageCode, localeObject)** - 允许你向 Scheduler 添加自定义语言环境
    - **languageCode** - (*string*) 两字母语言代码
    - **localeObject** - (*object*) 语言环境数据对象
- **setLocale(localeObject: string|object)** - 设置当前激活的语言环境

你可以传入语言代码:

~~~js
scheduler.i18n.setLocale("fr");
~~~

或者传入语言环境对象:

~~~js
scheduler.i18n.setLocale({
       labels: {
        day_tab: "Day",
   }
});
~~~

- **getLocale(languageCode : string)** - 通过语言代码获取语言环境对象
