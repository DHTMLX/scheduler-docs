# 迷你日历模板

![mini_calendar_templates](/img/mini_calendar_templates.png)

- [calendar_date](api/template/calendar_date.md) - 定义每个单元格中日期的显示方式
- [calendar_month](api/template/calendar_month.md) - 设置日历标题中显示的日期格式
- [calendar_scale_date](api/template/calendar_scale_date.md) - 控制周子标题中显示的星期名称
- [calendar_time](api/template/calendar_time.md) - 指定在 lightbox 中起始和结束日期输入框使用的日期格式
