# 视图

该库提供了10种完全可自定义的视图，方便您创建不同类型的日历，以吸引人的方式展示数据，并高效地进行管理。


<h2>默认视图</h2>

调度器自带了默认视图。这些视图以常见的格式组织您的事件:按天、按周或按月显示。

  <div>[![week](/img/week.png)](../week-view.md) [Week](views/week.md)</div>
  <div>[![month](/img/month.png)](../month-view.md) [Month](views/month.md)</div>
</div>

<h2>扩展视图</h2>
扩展视图为组织日历数据提供了更多选项:以表格形式展示事件、按资源分组并以垂直或水平方式显示分组、在 Google Maps 上显示地点等。
