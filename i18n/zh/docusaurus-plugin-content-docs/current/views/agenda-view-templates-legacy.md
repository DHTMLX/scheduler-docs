# 议程视图模板

*本文适用于 dhtmlxScheduler 6.0 及更早版本。关于 dhtmlxScheduler 7.0 及以上版本的信息，请参见[此处](views/agenda-view-templates.md)。*

![agenda_view_templates](/img/agenda_view_templates.png)

- [agenda_date](api/template/agenda_date.md) - 控制视图标题中显示的日期  
- [agenda_text](api/template/agenda_text.md) - 定义第二列中显示的文本  
- [agenda_time](api/template/agenda_time.md) - 指定第一列的日期格式
