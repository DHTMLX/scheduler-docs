# 议程视图模板

*对于使用 dhtmlxScheduler 6.0 或更早版本的用户，可以在[这里](views/agenda-view-templates-legacy.md)找到更多信息。

- [agenda_date](api/template/agenda_date.md) - 定义视图标题中显示的日期
- [agenda_text](api/template/agenda_text.md) - 控制事件标题的显示方式
- [agenda_time](api/template/agenda_time.md) - 设置事件时间的显示格式
- [agenda_day](api/template/agenda_day.md) - 自定义日期标签的外观
