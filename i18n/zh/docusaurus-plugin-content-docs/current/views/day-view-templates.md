# 日视图模板

![day_view_templates](/img/day_view_templates.png)

- [day_date](api/template/day_date.md) - 在头部显示日期  
- [day_scale_date](api/template/day_scale_date.md) - 在子头部显示日期  
- [event_class](api/template/event_class.md) - 定义应用于事件容器的CSS样式  
- [event_header](api/template/event_header.md) - 表示事件的头部区域  
- [event_text](api/template/event_text.md) - 包含事件的文本内容  
- [hour_scale](api/template/hour_scale.md) - 渲染Y轴上的项目
