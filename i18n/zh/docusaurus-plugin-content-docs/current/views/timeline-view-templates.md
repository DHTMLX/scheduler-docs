# 时间线视图模板

![timeline_view_templates_01](/img/timeline_view_templates_01.png)


![timeline_view_templates_02](/img/timeline_view_templates_02.png)

- [event_bar_text](api/template/event_bar_text.md) - 显示事件内部的文本
- [event_class](api/template/event_class.md) - 定义事件容器的CSS样式
- [`TIMELINE_cell_value`](api/template/timelinename_cell_value.md) - 显示视图单元格内已排程事件的数量
- [`TIMELINE_cell_class`](api/template/timelinename_cell_class.md) - 应用于单元格的CSS样式
- [`TIMELINE_scalex_class`](api/template/timelinename_scalex_class.md) - 设置X轴项目的CSS样式
- [`TIMELINE_second_scalex_class`](api/template/timelinename_second_scalex_class.md) - 使用CSS为第二X轴的项目设置样式
- [`TIMELINE_scaley_class`](api/template/timelinename_scaley_class.md) - 控制Y轴项目的CSS样式
- [`TIMELINE_scale_label`](api/template/timelinename_scale_label.md) - 表示Y轴上的标签
- [`TIMELINE_tooltip`](api/template/timelinename_tooltip.md)- 显示包含一个或多个排程事件的单元格的提示信息
- [`TIMELINE_date`](api/template/timelinename_date.md)- 在视图标题中显示日期
- [`TIMELINE_row_class`](api/template/timelinename_row_class.md) - 指定视图中某一行分配的CSS类
- [`TIMELINE_scale_date`](api/template/timelinename_scale_date.md)  - 表示X轴上的项目
- [`TIMELINE_second_scale_date`](api/template/timelinename_second_scale_date.md) - 表示第二X轴上的项目
