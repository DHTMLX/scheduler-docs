# 年视图模板

![year_view_templates](/img/year_view_templates.png)

- [event_class](api/template/event_class.md) - 定义事件容器的 CSS 样式
- [month_day](api/template/month_day.md) - 指定单元格内日期的显示方式
- [year_date](api/template/year_date.md) - 在视图头部显示日期
- [year_month](api/template/year_month.md) - 在每个月块的头部显示月份名称
- [year_scale_date](api/template/year_scale_date.md) - 在月份块的子头部显示星期名称
- [year_tooltip](api/template/year_tooltip.md) - 为包含已排定事件的日期单元格提供工具提示
