# 地图视图模板

![map_view_templates](/img/map_view_templates.png)

- [map_date](api/template/map_date.md) - 在视图的标题栏显示日期
- [map_text](api/template/map_text.md) - 在视图的第二列显示文本
- [map_time](api/template/map_time.md) - 在视图的第一列显示日期
- [map_info_content](api/template/map_info_content.md) - 定义地图视图中信息窗口的内容
