# Grid 视图模板

![grid_view_templates](/img/grid_view_templates.png)

- [event_class](api/template/event_class.md) - 定义应用于网格行的 CSS 样式
- [`{gridname}_date_template`](api/template/gridname_date.md) - 控制视图标题中日期的显示方式
- [`{gridname}_full_date_template`](api/template/gridname_full_date.md) - 指定 id='date' 列中使用的日期格式
- [`{gridname}_single_date_template`](api/template/gridname_single_date.md)  - 设置 id='start_date' 或 id='end_date' 列的日期格式
- [`{gridname}_field_template`](api/template/gridname_field.md) - 决定网格列中显示的文本
