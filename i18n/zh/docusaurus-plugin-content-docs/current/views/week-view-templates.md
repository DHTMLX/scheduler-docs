# 周视图模板

![week_view_templates](/img/week_view_templates.png)

- [event_class](api/template/event_class.md) - 定义事件容器的 CSS 样式
- [event_header](api/template/event_header.md) - 表示事件的头部区域
- [event_text](api/template/event_text.md) - 显示事件的主要文本内容
- [hour_scale](api/template/hour_scale.md) - 在 Y 轴上显示项目
- [week_date](api/template/week_date.md) - 在视图头部展示日期
- [week_date_class](api/template/week_date_class.md) - 设置单个日期单元格的 CSS 样式
- [week_scale_date](api/template/week_scale_date.md) - 在视图的子头部显示日期
