# 月视图模板

![month_view_templates](/img/month_view_templates.png)

- [event_bar_date](api/template/event_bar_date.md) - 显示事件的日期，仅用于单日事件
- [event_bar_text](api/template/event_bar_text.md) - 显示事件文本，适用于所有事件
- [event_class](api/template/event_class.md) - 定义事件容器的CSS样式，适用于单日和多日事件
- [month_date](api/template/month_date.md) - 表示视图标题栏中显示的日期
- [month_date_class](api/template/month_date_class.md) - 为单元格中的每一天应用CSS样式
- [month_day](api/template/month_day.md) - 控制单元格内日期的显示格式
- [month_events_link](api/template/month_events_link.md) - 管理单元格内的"查看更多"链接，适用于单日和多日事件
- [month_scale_date](api/template/month_scale_date.md) - 指定视图周标题栏中的星期名称
