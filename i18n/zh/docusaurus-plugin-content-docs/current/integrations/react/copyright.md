# 第三方许可

本页列出作为 **DHTMLX React Scheduler** 一部分分发的组件所包含的第三方软件告知与许可条款。

DHTMLX React Scheduler 是一个捆绑发行版，其中包含 DHTMLX Scheduler JavaScript 库。因此，与 DHTMLX Scheduler 重新分发的所有第三方组件也同样适用于 React Scheduler。

本页还列出特定于 React 封装器以及可选的 React 示例应用程序的第三方组件。
