# Map View 템플릿

![map_view_templates](/img/map_view_templates.png)

- [map_date](api/template/map_date.md) - 뷰의 헤더에 날짜를 표시합니다.
- [map_text](api/template/map_text.md) - 뷰의 두 번째 열에 텍스트를 보여줍니다.
- [map_time](api/template/map_time.md) - 뷰의 첫 번째 열에 날짜를 표시합니다.
- [map_info_content](api/template/map_info_content.md) - Map View에서 정보 창의 내용을 정의합니다.
