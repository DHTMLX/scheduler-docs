# 주간 뷰 템플릿

![week_view_templates](/img/week_view_templates.png)

- [event_class](api/template/event_class.md) - 이벤트 컨테이너의 CSS 스타일을 정의합니다.
- [event_header](api/template/event_header.md) - 이벤트의 헤더 섹션을 나타냅니다.
- [event_text](api/template/event_text.md) - 이벤트의 주요 텍스트 내용을 표시합니다.
- [hour_scale](api/template/hour_scale.md) - Y축에 항목을 표시합니다.
- [week_date](api/template/week_date.md) - 뷰의 헤더에 날짜를 표시합니다.
- [week_date_class](api/template/week_date_class.md) - 개별 요일 셀의 CSS 스타일을 설정합니다.
- [week_scale_date](api/template/week_scale_date.md) - 뷰의 서브 헤더에 날짜를 표시합니다.
