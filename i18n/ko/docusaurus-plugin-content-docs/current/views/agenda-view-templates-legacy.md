# Agenda View Templates

*이 문서는 dhtmlxScheduler 6.0 및 이전 버전에 대한 내용을 다룹니다. dhtmlxScheduler 7.0 이상 버전에 대한 정보는 [여기](views/agenda-view-templates.md)를 참고하세요.*

![agenda_view_templates](/img/agenda_view_templates.png)

- [agenda_date](api/template/agenda_date.md) - 뷰 헤더에 표시되는 날짜를 제어합니다.
- [agenda_text](api/template/agenda_text.md) - 두 번째 열에 표시되는 텍스트를 정의합니다.
- [agenda_time](api/template/agenda_time.md) - 첫 번째 열에 사용할 날짜 형식을 지정합니다.
