# 타임라인 뷰 템플릿

![timeline_view_templates_01](/img/timeline_view_templates_01.png)


![timeline_view_templates_02](/img/timeline_view_templates_02.png)

- [event_bar_text](api/template/event_bar_text.md) - 이벤트 내부에 표시되는 텍스트를 설정합니다.
- [event_class](api/template/event_class.md) - 이벤트 컨테이너의 CSS 스타일을 정의합니다.
- [`TIMELINE_cell_value`](api/template/timelinename_cell_value.md) - 뷰의 셀 안에 예약된 이벤트의 개수를 표시합니다.
- [`TIMELINE_cell_class`](api/template/timelinename_cell_class.md) - 셀에 CSS 스타일을 적용합니다.
- [`TIMELINE_scalex_class`](api/template/timelinename_scalex_class.md)- X축 아이템에 대한 CSS 스타일을 설정합니다.
- [`TIMELINE_second_scalex_class`](api/template/timelinename_second_scalex_class.md) - 두 번째 X축의 아이템을 CSS로 스타일링합니다.
- [`TIMELINE_scaley_class`](api/template/timelinename_scaley_class.md)  - Y축 아이템에 대한 CSS 스타일을 제어합니다.
- [`TIMELINE_scale_label`](api/template/timelinename_scale_label.md) - Y축 라벨을 나타냅니다.
- [`TIMELINE_tooltip`](api/template/timelinename_tooltip.md) - 하나 이상의 예약된 이벤트가 포함된 셀에 대한 툴팁을 표시합니다.
- [`TIMELINE_date`](api/template/timelinename_date.md)- 뷰 헤더에 날짜를 표시합니다.
- [`TIMELINE_row_class`](api/template/timelinename_row_class.md) - 뷰의 행에 할당된 CSS 클래스를 지정합니다.
- [`TIMELINE_scale_date`](api/template/timelinename_scale_date.md) - X축의 아이템을 나타냅니다.
- [`TIMELINE_second_scale_date`](api/template/timelinename_second_scale_date.md) - 두 번째 X축의 아이템을 나타냅니다.
