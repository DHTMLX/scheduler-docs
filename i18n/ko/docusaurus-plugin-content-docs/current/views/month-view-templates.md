# Month View Templates

![month_view_templates](/img/month_view_templates.png)

- [event_bar_date](api/template/event_bar_date.md) - 이벤트의 날짜를 표시하며, 주로 하루짜리 이벤트에 사용됩니다.
- [event_bar_text](api/template/event_bar_text.md) - 이벤트의 텍스트를 표시하며, 모든 이벤트에 적용됩니다.
- [event_class](api/template/event_class.md) - 이벤트 컨테이너의 CSS 스타일을 정의하며, 하루짜리와 다일 이벤트 모두에 사용됩니다.
- [month_date](api/template/month_date.md) - 뷰 헤더에 표시되는 날짜를 나타냅니다.
- [month_date_class](api/template/month_date_class.md) - 개별 날짜 셀에 CSS 스타일을 적용합니다.
- [month_day](api/template/month_day.md) - 셀 내에 표시되는 날짜 형식을 제어합니다.
- [month_events_link](api/template/month_events_link.md) - 셀 내의 'View more' 링크를 관리하며, 하루짜리와 다일 이벤트 모두에 사용됩니다.
- [month_scale_date](api/template/month_scale_date.md) - 뷰의 주 헤더에 표시되는 요일 이름을 지정합니다.
