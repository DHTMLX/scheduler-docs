# Year View Templates

![year_view_templates](/img/year_view_templates.png)

- [event_class](api/template/event_class.md) - 이벤트 컨테이너의 CSS 스타일을 정의합니다.
- [month_day](api/template/month_day.md) - 셀 내부에서 일(day)이 표시되는 방식을 지정합니다.
- [year_date](api/template/year_date.md) - 뷰 헤더에 날짜를 표시합니다.
- [year_month](api/template/year_month.md) - 각 월(month) 블록의 헤더에 월 이름을 표시합니다.
- [year_scale_date](api/template/year_scale_date.md) - 월 블록 내의 서브 헤더에 요일 이름을 표시합니다.
- [year_tooltip](api/template/year_tooltip.md) - 일정이 있는 일(day) 셀에 툴팁을 제공합니다.
