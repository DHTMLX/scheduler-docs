# Day View Templates

![day_view_templates](/img/day_view_templates.png)

- [day_date](api/template/day_date.md) - 헤더에 날짜를 표시합니다.  
- [day_scale_date](api/template/day_scale_date.md) - 서브 헤더에 날짜를 보여줍니다.  
- [event_class](api/template/event_class.md) - 이벤트 컨테이너에 적용되는 CSS 스타일을 정의합니다.  
- [event_header](api/template/event_header.md) - 이벤트의 헤더 섹션을 나타냅니다.  
- [event_text](api/template/event_text.md) - 이벤트의 텍스트 내용을 포함합니다.  
- [hour_scale](api/template/hour_scale.md) - Y축을 따라 항목을 렌더링합니다.
