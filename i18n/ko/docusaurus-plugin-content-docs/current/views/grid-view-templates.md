# 그리드 뷰 템플릿

![grid_view_templates](/img/grid_view_templates.png)

- [event_class](api/template/event_class.md) - 그리드 행에 적용되는 CSS 스타일을 정의합니다.
- [`{gridname}_date_template`](api/template/gridname_date.md)- 뷰 헤더에서 날짜 표시를 제어합니다.
- [`{gridname}_full_date_template`](api/template/gridname_full_date.md) - id='date'인 열에서 사용되는 날짜 형식을 지정합니다.
- [`{gridname}_single_date_template`](api/template/gridname_single_date.md) - id='start_date' 또는 id='end_date'인 열의 날짜 형식을 설정합니다.
- [`{gridname}_field_template`](api/template/gridname_field.md) - 그리드 열에 표시되는 텍스트를 결정합니다.
