# Agenda View Templates 

*dhtmlxScheduler 6.0 이하 버전을 사용하는 경우, 추가 정보는 [여기](views/agenda-view-templates-legacy.md)에서 확인할 수 있습니다.

- [agenda_date](api/template/agenda_date.md) - 뷰의 헤더에 표시되는 날짜를 정의합니다.
- [agenda_text](api/template/agenda_text.md) - 이벤트 제목이 표시되는 방식을 제어합니다.
- [agenda_time](api/template/agenda_time.md) - 이벤트 시간 형식을 설정합니다.
- [agenda_day](api/template/agenda_day.md) - 날짜 레이블의 표시 방식을 사용자 정의합니다.
