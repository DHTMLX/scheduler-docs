# AI 도구

이 섹션은 두 가지 방법으로 DHTMLX Scheduler에서 AI를 사용하는 자료를 수집합니다:

- **AI 보조 개발** - 앱 빌더, 코딩 에이전트 및 도구 통합.
- **앱 내 AI 기능** - 데모 및 권장 구현 패턴.

## MCP 서버

- [DHTMLX MCP 서버](./mcp-server/) — 최신 DHTMLX 문서 및 API 참조에 AI 도구를 연결합니다.
