# tooltip

### Description

@short: 이벤트에 대한 tooltip을 표시합니다.

@signature: tooltip: any

### Example

~~~jsx
var tooltip = scheduler.tooltip;
tooltip.hide();
tooltip.show(event, text);
~~~

### Details

:::note
 이 옵션은 [tooltip](guides/extensions-list.md#tooltip) 플러그인이 활성화된 경우에만 작동합니다. 
:::
