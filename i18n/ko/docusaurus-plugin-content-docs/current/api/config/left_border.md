# left_border

### Description

@short: 스케줄러에서 점선 왼쪽 테두리를 활성화합니다

@signature: left_border: boolean

### Example

~~~jsx
scheduler.config.left_border = true;
...
scheduler.init('scheduler_here', new Date(2013, 7, 5), "month");
~~~

**Default value:** false
