# touch_swipe_dates

### Description

@short: 모바일 기기에서 가로 스와이프 제스처로 활성 날짜를 전환하는 기능을 켜거나 끌 수 있습니다.

@signature: touch_swipe_dates: boolean

### Example

~~~jsx
scheduler.config.touch_swipe_dates = true;
~~~

**Default value:** false
