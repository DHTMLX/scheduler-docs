# touch_tooltip

### Description

@short: DhtmlxScheduler의 툴팁을 터치 디바이스에서 표시할 수 있도록 활성화합니다.

@signature: touch_tooltip: boolean

### Example

~~~jsx
// 터치 디바이스에서 툴팁을 활성화합니다.
scheduler.config.touch_tooltip = true;
~~~

**Default value:** false
