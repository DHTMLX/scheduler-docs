# show_quick_info

### Description

@short: 'quick_info' 확장 기능(작업 세부 정보를 보여주는 팝업)을 켜거나 끕니다.

@signature: show_quick_info: boolean

### Example

~~~jsx
scheduler.config.show_quick_info = false;
scheduler.init("scheduler_here");
~~~

**Default value:** true
