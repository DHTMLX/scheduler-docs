# collapse

### Description

@short: 스케줄러를 확장된 상태에서 일반 크기로 되돌립니다.

@signature: collapse: () =\> void

### Example

~~~jsx
scheduler.collapse();
~~~

### Details

:::note
 이 메서드는 [expand](guides/extensions-list.md#expand) 확장 기능이 활성화되어 있어야 작동합니다. 
:::
