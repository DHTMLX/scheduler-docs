# 서드파티 라이선스

이 문서에서는 DHTMLX Scheduler 라이브러리에 포함된 서드파티 소프트웨어 컴포넌트에 대한 고지 및 추가 조건을 안내합니다.

**Roboto Google 폰트**는 기본적으로 라이브러리에 포함되어 있지 않습니다. Scheduler v5.0부터 Material 스킨은 `https://fonts.googleapis.com`에서 Roboto 폰트를 동적으로 로드합니다. 이 폰트는 Apache Software License로 라이선스가 부여되어 있습니다.

DHTMLX Scheduler 라이브러리는 Scheduler 스킨의 소스 LESS 파일을 제공합니다. NPM 패키지를 설치하고 *codebase/sources/less/package.json* 파일의 npm-watch 스크립트를 사용하여 스킨을 커스터마이즈하고 재빌드할 수 있습니다. 이 목적을 위해 사용되는 **npm-watch** 및 **less** 라이브러리는 각각 MIT 및 Apache 라이선스 하에 배포되며, 기본적으로 scheduler 패키지에 포함되어 있지 않습니다.

또한, 샘플 애플리케이션에서는 **DHTMLX Suite 5.x** 및 **DHTMLX Suite 6.x**와 같은 오픈 소스 라이브러리가 사용됩니다. 샘플에는 **Underscore** 및 **Backbone** Node.js 모듈도 포함되어 있습니다. **jQuery** 라이브러리는 CDN에서 로드됩니다. 백엔드 샘플에서는 **Express, body-parser, helmet** 등과 같은 Node.js 모듈을 사용합니다.
