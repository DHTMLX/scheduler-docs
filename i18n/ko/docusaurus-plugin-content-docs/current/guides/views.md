# Views

이 라이브러리는 10가지 완전히 커스터마이즈 가능한 뷰를 제공하여 다양한 유형의 캘린더를 손쉽게 만들고, 데이터를 매력적으로 표시하며 효율적으로 관리할 수 있도록 도와줍니다.


<h2>기본 뷰</h2>

스케줄러에는 기본 뷰가 이미 포함되어 있습니다. 이 뷰들은 일, 주, 월 등 일반적인 형식으로 이벤트를 정리해 보여줍니다.

<h2>확장 뷰</h2>
확장 뷰를 통해 캘린더 데이터를 다양한 방식으로 구성할 수 있습니다. 예를 들어, 이벤트를 표 형식으로 보여주거나, 자원별로 그룹화하여 수직 또는 수평으로 그룹을 표시하거나, Google Maps에 위치를 표시하는 등 다양한 옵션을 제공합니다.
