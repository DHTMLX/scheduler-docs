# 날짜 형식 지정

날짜 형식을 설정할 때 다음 문자들을 사용할 수 있습니다:


- **%y** - 연도를 두 자리 숫자로 표시합니다 ( _00에서 99_ );
- **%Y** - 연도를 네 자리 숫자로 표시합니다 ( _1900-9999_ );


- **%m** - 월을 앞에 0이 붙은 두 자리 숫자로 표시합니다 ( _01에서 12_ );
- **%n** - 월을 앞에 0 없이 숫자로 표시합니다 ( _1에서 12_ );
- **%M** - 월을 축약형으로 표시합니다 ( _Jan에서 Dec_ );
- **%F** - 월의 전체 이름을 표시합니다 ( _January에서 December_ );


- **%W** - ISO-8601 기준으로 연도의 주 번호를 표시하며, 주는 월요일부터 시작합니다;


- **%d** - 일을 앞에 0이 붙은 두 자리 숫자로 표시합니다 ( _01에서 31_ );
- **%j** - 일을 앞에 0 없이 숫자로 표시합니다 ( _1에서 31_ );
- **%D** - 요일을 축약형으로 표시합니다 ( _Sun에서 Sat_ );
- **%l** - 요일의 전체 이름을 표시합니다 ( _Sunday에서 Saturday_ );


- **%h** - 12시간제로 시를 표시합니다 ( _00에서 11_ );
- **%H** - 24시간제로 시를 표시합니다 ( _00에서 23_ );
- **%g** - 앞에 0 없이 12시간제로 시를 표시합니다 ( _1에서 12_ );
- **%G** - 앞에 0 없이 24시간제로 시를 표시합니다 ( _0에서 23_ );


- **%i** - 분을 앞에 0이 붙은 두 자리 숫자로 표시합니다 ( _00에서 59_ );
- **%s** - 초를 앞에 0이 붙은 두 자리 숫자로 표시합니다 ( _00에서 59_ );
- **%a** - **am**(자정부터 정오까지)과 **pm**(정오부터 자정까지)을 표시합니다;
- **%A** - **AM**(자정부터 정오까지)과 **PM**(정오부터 자정까지)을 표시합니다.


예를 들어, 2013년 6월 1일을 01/06/2013 형태로 표시하려면 "%d/%m/%Y" 형식을 사용하면 됩니다.
