# Mini Calendar Templates

![mini_calendar_templates](/img/mini_calendar_templates.png)

- [calendar_date](api/template/calendar_date.md) - 각 셀 안에 표시되는 날짜의 모양을 정의합니다.
- [calendar_month](api/template/calendar_month.md) - 캘린더 헤더에 표시되는 날짜 형식을 설정합니다.
- [calendar_scale_date](api/template/calendar_scale_date.md) - 주간 서브 헤더에 표시되는 요일 이름을 제어합니다.
- [calendar_time](api/template/calendar_time.md) - 라이트박스에서 시작 및 종료 날짜 입력에 사용되는 날짜 형식을 지정합니다.
