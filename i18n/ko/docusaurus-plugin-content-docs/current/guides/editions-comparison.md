# Standard vs PRO 라이브러리 버전 비교

dhtmlxScheduler 라이브러리의 Standard 버전과 PRO 버전의 차이점이 궁금하시거나, PRO 에디션에서 제공하는 추가 기능이 무엇인지 알고 싶으시다면 이 가이드에서 모든 정보를 확인하실 수 있습니다.

PRO 에디션에는 Standard 버전의 모든 기능이 포함되어 있으며, 여기에 몇 가지 유용한 추가 기능이 더해집니다. 아래 표는 각 기능을 나열하여 dhtmlxScheduler의 두 가지 버전을 쉽게 비교할 수 있도록 도와줍니다.
