drag_event_body
=============

@short: allows dragging scheduler events by any part of the body

	

@type: boolean
@default:true
@example:
scheduler.config.drag_event_body = false;

@template:	api_config
@descr:
If the config is disabled, drag-n-drop is possible only by the header of the event.
