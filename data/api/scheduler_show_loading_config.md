show_loading
=============
@short:enables showing a progress/spinner while data is loading (useful for dynamic loading)
	

@type: boolean
@default:false
@example:
scheduler.config.show_loading = true;
...
scheduler.init('scheduler_here',new Date(2013,0,10),"week");
