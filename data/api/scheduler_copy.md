copy
=============


@short:
	creates a deep copy of the provided object

@params:

- event		object		the object that needs to be copied

@returns:

- backupEvent		object		a deep copy of the provided object

@example:

var backupEvent = scheduler.copy(scheduler.getEvent(id));

@descr:
