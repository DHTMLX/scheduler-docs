# Week View Templates

![week_view_templates](/img/week_view_templates.png)

- [event_class](api/template/event_class.md) - the CSS style for the event container
- [event_header](api/template/event_header.md) - the header of an event
- [event_text](api/template/event_text.md) - the text of an event
- [hour_scale](api/template/hour_scale.md) - the items of the Y-Axis
- [week_date](api/template/week_date.md) - the date in the header of the view
- [week_date_class](api/template/week_date_class.md) - the CSS style for a day cell
- [week_scale_date](api/template/week_scale_date.md) - the date in the sub-header of the view
