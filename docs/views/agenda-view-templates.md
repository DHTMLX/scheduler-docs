# Agenda View Templates

*If you use dhtmlxScheduler 6.0 or earlier, see details [here](views/agenda-view-templates-legacy.md).

- [agenda_date](api/template/agenda_date.md) - the date in the header of the view
- [agenda_text](api/template/agenda_text.md) - the template for the event title
- [agenda_time](api/template/agenda_time.md) - the template for the event hours
- [agenda_day](api/template/agenda_day.md) - the template for the date label
