# Month View Templates

![month_view_templates](/img/month_view_templates.png)

- [event_bar_date](api/template/event_bar_date.md) - the date of an event. Applied to one-day events
- [event_bar_text](api/template/event_bar_text.md) - the text of an event. Applied to all events
- [event_class](api/template/event_class.md) - the CSS style for the event container. Applied to both one-day and multiday events
- [month_date](api/template/month_date.md) - the date in the header of the view
- [month_date_class](api/template/month_date_class.md)  - the CSS style for a day cell
- [month_day](api/template/month_day.md) - the format of the day in a cell
- [month_events_link](api/template/month_events_link.md) - the 'View more' link in the cell. Applied to both one-day and multiday events 
- [month_scale_date](api/template/month_scale_date.md) - the name of the day in the week header of the view
