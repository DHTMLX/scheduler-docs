# collapse

### Description

@short: Collapses the expanded scheduler back to the normal size

@signature: collapse: () =\> void

### Example

~~~jsx
scheduler.collapse();
~~~

### Details

:::note
 The method requires the [expand](guides/extensions-list.md#expand) extension to be enabled. 
:::
