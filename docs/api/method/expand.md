# expand

### Description

@short: Expands the scheduler to the full screen view

@signature: expand: () =\> void

### Example

~~~jsx
scheduler.expand();
~~~

### Details

:::note
 The method requires the [expand](guides/extensions-list.md#expand) plugin to be enabled.
