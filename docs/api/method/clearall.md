# clearAll

### Description

@short: Removes all events from the scheduler

@signature: clearAll: () =\> void

### Example

~~~jsx
scheduler.clearAll();
~~~

### Details

Note, the method invokes the [onClearAll](api/event/onclearall.md) event.
