# tooltip

### Description

@short: Displays tooltips for events

@signature: tooltip: any

### Example

~~~jsx
var tooltip = scheduler.tooltip;
tooltip.hide();
tooltip.show(event, text);
~~~

### Details

:::note
 The option requires the [tooltip](guides/extensions-list.md#tooltip) plugin to be enabled. 
:::
