# touch_swipe_dates

### Description

@short: Enables/disables switching active dates using the horizontal swipe gesture on mobiles

@signature: touch_swipe_dates: boolean

### Example

~~~jsx
scheduler.config.touch_swipe_dates = true;
~~~

**Default value:** false
