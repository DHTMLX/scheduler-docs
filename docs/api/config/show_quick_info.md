# show_quick_info

### Description

@short: Activates/disables the 'quick_info' extension (pop-up task's details form)

@signature: show_quick_info: boolean

### Example

~~~jsx
scheduler.config.show_quick_info = false;
scheduler.init("scheduler_here");
~~~

**Default value:** true
