# touch_tooltip

### Description

@short: Allows displaying dhtmxlScheduler's tooltips on the touch devices

@signature: touch_tooltip: boolean

### Example

~~~jsx
// enables displaying tooltips on the touch devices
scheduler.config.touch_tooltip = true;
~~~

**Default value:** false
